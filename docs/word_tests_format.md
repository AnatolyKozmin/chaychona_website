# Импорт тестов из Word (.docx)

Файл должен быть в формате **.docx** (Word 2007+).

## Структура

1. Каждый вопрос начинается с **номера и точки**: `1.`, `2.`, `3.` …
2. **Текст вопроса** — всё от номера до первого варианта ответа вида **`A)`** или **`а)`** (латинская **A** или кириллическая **а**).
3. **Варианты ответа** начинаются с буквы и скобки: `A)`, `B)`, `C)` … (латиница или кириллица).
4. Варианты можно писать **в одной строке** с вопросом или **каждый с новой строки** (отдельный абзац).
5. **Правильные ответы** в Word нужно выделить **жирным** (только текст варианта после `X)`). Неверные — обычным начертанием.

## Пример

```
1. Какой соус подаётся к шашлыку?
A) Ткемали
B) Майонез
C) Соевый
```

Если правильный ответ «Ткемали», выделите **Ткемали** жирным.

## Несколько правильных ответов

Выделите жирным все верные варианты — тест будет импортирован как вопрос с типом **multiple**.

## Импорт в систему

В интерфейсе «Конструктор тестов» укажите код теста, название, ресторан и роль, затем загрузите `.docx`.  
Импорт из Excel по-прежнему доступен для табличного формата.
